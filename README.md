# MM804-Point-Cloud-Visualization

## Dependencies: 
> enaml                         0.15.2

> vtk                           9.2.6

> Python 3.9.13
### Install Enaml
``` 
conda install enaml -c conda-forge
```

### Install VTK
``` 
pip install vtk
```

## Run the Project 
```Python3
python3 MM804_Point_Cloud_Vis_Project.py
```
